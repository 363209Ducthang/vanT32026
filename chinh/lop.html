Mục tiêu khoá học

Khóa học hướng dẫn các kiến thức cơ bản về lập trình web, sử dụng HTML, CSS và JavaScript để xây dựng trang web đơn giản. Học viên sẽ tìm hiểu cách tạo cấu trúc trang, áp dụng kiểu dáng và thêm tương tác cơ bản.

Nội dung chính

  • Giới thiệu HTML và cấu trúc tài liệu
  • CSS cơ bản và layout trang
  • JavaScript nền tảng và DOM
  • Thực hành dự án nhỏ
